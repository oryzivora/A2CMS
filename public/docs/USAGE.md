## 快速开始

### 添加角色

1. 点击页面顶部的 **「+」** 按钮
2. 填写角色基本信息（名称、职业、种族、服务器）
3. 点击确认添加

### 编辑角色

- **快速编辑**：点击角色卡片右上角的编辑图标
- **直接编辑**：双击角色卡片中的能量值或其他数值

### 删除角色

1. 点击角色卡片上的删除图标
2. 如需防止误删，可先点击锁定图标 🔒 锁定角色

## 角色管理

### 视图切换

右上角提供 **卡片视图** 和 **表格视图** 切换按钮。

### 布局调整

- 支持 2/3/4/5 列网格布局
- 通过布局按钮组进行调整

### 角色数据

| 字段 | 说明 |
|------|------|
| 能量 | 奥德能量，上限 840（可额外补充至 2000） |
| 远征 | 每日 5:00/17:00 各恢复 1 次，上限 14 |
| 超越 | 每日 5:00/17:00 各恢复 1 次，上限 7 |
| 噩梦 | 每日 5:00 恢复 +2，上限 14（账号共享） |
| 树古 | 每日 5:00 恢复（会员+2/非会员+1），上限 14 |
| 卢德莱/净化所 | 圣域副本，每周周三 5:00 重置 |

### 任务状态

点击任务按钮可标记完成/未完成状态：

| 任务 | 重置时间 |
|------|----------|
| 每日签到 | 每日 0:00 |
| 深渊回廊 | 每周三/周六 22:00（韩服 21:00） |
| 觉醒战 | 每周周三 5:00 |
| 卢德莱/净化所 | 每周周三 5:00 |
| 深渊指令书/本地指令书 | 每周周三 5:00 |

## 账号系统

### 添加账号

1. 点击账号栏的 **「+」** 按钮
2. 填写账号名称、选择种族和服务器
3. 点击确认

### 账号维度数据

以下数据在同一账号的所有角色间共享：

- 每日副本（入场+补充次数）
- 树古庆典（奖励钥匙+补充次数）
- 噩梦挑战（挑战+补充次数）
- 深渊指令书（完成状态）
- 本地指令书（本周次数/上限）
- 每日使命（今日次数/上限）
- 商店奥德（完成次数）
- 转换奥德（完成次数）

### 切换账号

点击账号下拉框可切换当前账号，角色列表会自动筛选。

## 定时任务

系统会在打开时自动计算并应用定时任务。

### 能量恢复

| 会员 | 非会员 | 上限 |
|------|--------|------|
| +15/3小时 | +10/3小时 | 840 |

### 每日重置

| 时间 | 内容 |
|------|------|
| 0:00 | 每日使命重置 |
| 5:00 | 远征+1、超越+1、噩梦+2、树古+2(会员)/+1(非会员) |
| 17:00 | 远征+1、超越+1 |

### 每周重置（周三 5:00）

- 圣域卢德莱/净化所奖励+挑战重置
- 深渊指令书/本地指令书/觉醒战重置
- 每日副本次数重置为上限
- 每周指令重置为上限
- 商店奥德/转换奥德重置为上限

### 深渊回廊重置

| 服务器 | 重置时间 |
|--------|----------|
| 国服 | 每周三/周六 22:00 |
| 韩服 | 每周三/周六 21:00 |

## 数据管理

### 自动保存

所有数据自动保存到浏览器 localStorage，刷新或关闭页面不会丢失。

### 导出数据

1. 点击设置按钮 ⚙️
2. 选择 **「导出数据」**
3. 保存 JSON 文件

### 导入数据

1. 点击设置按钮 ⚙️
2. 选择 **「导入数据」**
3. 选择之前导出的 JSON 文件

### 切换服务器

右上角服务器选择器可切换 **国服 (CN)** / **韩服 (KR)**，切换后会根据服务器时间自动调整重置时间。

## 常见问题

### Q: 为什么能量没有自动恢复？

A: 请确保两次打开应用的时间间隔超过 3 小时。能量恢复在打开应用时计算。

### Q: 如何重置所有任务状态？

A: 在设置菜单中选择「清空任务」。

### Q: 数据存储在哪里？

A: 数据存储在浏览器 localStorage 中，清除浏览器数据会导致丢失。建议定期导出备份。

### Q: 如何在不同浏览器间同步数据？

A: 可使用「导出/导入」功能手动同步，或使用浏览器自带的书签/插件同步功能。
